# Configuring `rack-timeout` gem

## `RACK_TIMEOUT_SERVICE_TIMEOUT`

Default: `15`.

Default in container: as default.

## `RACK_TIMEOUT_WAIT_TIMEOUT`

Default: `30`.

Default in container: as default.

## `RACK_TIMEOUT_WAIT_OVERTIME`

Default: `60`.

Default in container: as default.

## `RACK_TIMEOUT_SERVICE_PAST_WAIT`

Default: `false`.

Default in container: as default.

## `RACK_TIMEOUT_TERM_ON_TIMEOUT`

Default: `0`.

Default in container: as default.
