# v0.10.9 Release notes

This is security and bug-fix release.

NOTE: This is release doesn't support JRuby anymore.

## Security

1. Ruby 3.4.9. [CVE-2026-27820 in zlib gem](https://www.ruby-lang.org/en/news/2026/03/11/ruby-3-4-9-released/)
2. Update `bcrypt` gem from 3.1.20 to 3.1.22. [CVE-2026-33306](https://github.com/bcrypt-ruby/bcrypt-ruby/security/advisories/GHSA-f27w-vcwj-c954)
3. Update `nokogiri` gem from 1.18.9 to 1.19.2. [GHSA-wx95-c6cv-8532](https://github.com/sparklemotion/nokogiri/security/advisories/GHSA-wx95-c6cv-8532)
4. Update `loofah` gem from 2.24.0 to 2.25.1. [GHSA-46fp-8f5p-pf2m](https://github.com/flavorjones/loofah/security/advisories/GHSA-46fp-8f5p-pf2m)
5. Update `devise` gem from 4.9.4 to 5.0.3. [CVE-2026-32700](https://github.com/heartcombo/devise/security/advisories/GHSA-57hq-95w6-v4fc)

[CVE-2026-33176](https://github.com/advisories/GHSA-2j26-frm8-cmj9), [CVE-2026-33170](https://github.com/advisories/GHSA-89vf-4333-qx8v), [CVE-2026-33169](https://github.com/advisories/GHSA-cg4j-q9v8-6v38), [CVE-2026-33168](https://github.com/advisories/GHSA-v55j-83pf-r9cq), [CVE-2026-33173](https://github.com/advisories/GHSA-qcfx-2mfw-w4cg), [CVE-2026-33174](https://github.com/advisories/GHSA-r46p-8f7g-vvvg), [CVE-2026-33658](https://github.com/advisories/GHSA-p9fm-f462-ggrg), [CVE-2026-33195](https://github.com/advisories/GHSA-9xrj-h377-fr87), [CVE-2026-33202](https://github.com/advisories/GHSA-73f9-jhhh-hr5m).
